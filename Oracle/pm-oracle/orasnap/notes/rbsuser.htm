ROLLBACK USER NOTES:
  • Segment Name - Name of the rollback segment.
  • Xacts - Number of active transactions.
  • SID - Session identifier.
  • Serial# - Session serial number. Used to uniquely identify a sessions' objects.
  • Username - Oracle username.
  • OS User - Operating system client user name.
  • SQL Text - Text of the SQL statement requiring the rollback segment, or the PL/SQL anonymous code.
