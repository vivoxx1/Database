TABLE LEVEL PARTITIONING NOTES:
  • Table Owner - Owner of the partitioned table
  • Table Name - Name of the partitioned table
  • Partition Name - Partition name
  • High Value Length - Length of the partition bound value expression
  • Partition Position - Position of the partition within the table
  • Tablespace - Name of the tablespace containing the partition
  • Pctfree - Minimum percentage of free space in a block
  • Pctused - Minimum percentage of used space in a block
  • Initrans - Initial number of transactions
  • Maxtrans - Maximum number of transactions
  • Initial Extent - Size of the initial extent in bytes
  • Next Extent - Size of secondary extents in bytes
  • Min Extents - Minimum number of extents allowed in the segments
  • Max Extents - Maximum number of extents allowed in the segments
  • Pct Increase - Percentage increase in extent size
  • Freelists - Number of process freelists allocated in this segment
  • Freelist Groups - Number of freelist groups allocated in this segment
  • Logging - Logging attribute of partition
  • Num Rows * - Number of rows in the partition
  • Blocks * - Number of used blocks in the partition
  • Empty Blocks * - Number of empty (never used) blocks in the partition
  • Avg Space * - Average available free space in the partition
  • Chain Count * - Number of chained rows in the partition
  • Avg Row Length * - Average row length, including row overhead
  • Sample Size * - Sample size used in analyzing this partition
  • Last Analyzed * - Date of the most recent time this parition was analyzed
  • Buffer Pool - The buffer pool for the partition

    * - Populated when table has been analyzed
