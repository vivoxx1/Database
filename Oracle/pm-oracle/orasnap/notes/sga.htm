SGA NOTES:
  • Name - SGA component group
  • Value - Memory size in bytes

  • This reports shows the SGA memory structures of your system.
  • Database Buffers (8i) - The value of the init.ora parameters DB_BLOCK_BUFFERS multiplied by DB_BLOCK_SIZE
  • Database Buffers (9i,10g) - The value of the init.ora parameters DB_CACHE_SIZE
  • Redo Buffers - The value of the init.ora parameter LOG_BUFFER
