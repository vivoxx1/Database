SESSION TIME NOTES:
  • SID - Session ID of the session
  • Serial# - Serial# of the session
  • Username - Oracle username
  • OS User - Name of the operating system user
  • Server - Server Type: DEDICATED, SHARED, PSEUDO, NONE, etc.
  • Status - Status of the session: ACTIVE (currently executing SQL), INACTIVE, KILLED (marked to be killed), CACHED (temporarily caches for use by Oracle*XA)
  • Type - Session type
  • Program - Operating system program name
  • Logon Time - Time session was started
  • Last Call - Last time session was active
