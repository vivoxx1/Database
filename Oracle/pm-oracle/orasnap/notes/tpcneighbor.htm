TWO PHASE NEIGHBOR NOTES:
  • Local Tran Id - Local identifier of a transaction
  • In/Out - IN for incoming connections; OUT for outgoing
  • Database - IN: client database name; OUT: outgoing database link
  • DBUser Owner - IN: name of local user; OUT: owner of database link
  • Interface - C for request commit, otherwise N for prepare or request readonly commit
  • DB ID - The database ID at the other end of the connection
  • Sess# - Session number of the connection at this database
  • Branch - Transaction branch ID of the connection at this database
