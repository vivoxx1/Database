LARGE OBJECT NOTES:
  • Owner - Owner of the object
  • Segment Name - Name of the segment
  • Segment Type - Type of segment
  • Tablespace Name - Name of the tablespace
  • Bytes - Size of the object
  • Initial Extent - Size of the initial extent (bytes)
  • Next Extent - Size of the next extent (bytes)
  • Extents - Current number of extents
  • Percent Increase - Percent increase for object

  • This reports shows objects larger than the average segment size of all database segments.
