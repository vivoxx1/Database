OBJECTS THAT CANNOT EXTEND NOTES:
  • Owner - Owner of the object
  • Object Name - Name of the object
  • Object Type - Type of object
  • Tablespace - Name of the tablespace
  • Next Bytes - Size of next extent
  • Max Bytes - Max extent size (bytes)
  • Sum Bytes - Sum of free space (bytes)
  • Current Extents - Number of current extents for object
