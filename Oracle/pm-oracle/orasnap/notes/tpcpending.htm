TWO PHASE PENDING NOTES:
  • Local Tran Id - Local identifier of a transaction
  • Global Tran ID - Globally unique transaction ID
  • State - Collecting, prepared, committed, forced commit, or forced rollback
  • Mixed - YES = part of the transaction committed and part rolled back
  • Advice - C for commit, R for rollback, else NULL
  • Tran Comment - Text for commit work comment text
  • Fail Time - Value of SYSDATE when the row was inserted (tx or system recovery)
  • Force Time - Time of manual force decision (null if not forced locally)
  • Retry Time - Time automatic recovery (RECO) last tried to recover the transaction
  • OS User - Time automatic recovery (RECO) last tried to recover the transaction
  • OS Terminal - Time automatic recovery (RECO) last tried to recover the transaction
  • Host - Name of the host machine for the end-user
  • DB User - Name of the host machine for the end-user
  • Commit# - Name of the host machine for the end-user
